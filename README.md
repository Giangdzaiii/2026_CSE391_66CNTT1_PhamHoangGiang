# Thực hành nền tảng Web

Mô tả ngắn gọn về dự án này.  
Repo dùng để lưu trữ các bài tập thực hành

##  Mục tiêu
- Thực hành kiến thức HTML,CSS,Javascript đã học
- Làm quen với cấu trúc project
- Rèn kỹ năng code và quản lý source code với Git

## Công nghệ sử dụng
- Ngôn ngữ: HTML,CSS,Javascript
- Framework / Library: ReactJS
- Công cụ khác: Git
